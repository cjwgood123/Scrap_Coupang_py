# 🚀 간단 실행 가이드 (Git Bash)

## 가상환경이 이미 있는 경우

```bash
# 1. 가상환경 활성화
source venv/Scripts/activate

# 2. 패키지 설치 (처음 한 번만, 또는 업데이트 필요시)
pip install -r requirements.txt

# 3. 실행
python coupang_scraper_final.py
```

## 가상환경이 없는 경우 (처음 설치)

```bash
# 1. 가상환경 생성
py -m venv venv

# 2. 가상환경 활성화
source venv/Scripts/activate

# 3. 패키지 설치
pip install -r requirements.txt

# 4. 실행
python coupang_scraper_final.py
```

## 다음번부터는 (가상환경이 있는 경우)

```bash
# 1. 가상환경 활성화만 하면 됩니다
source venv/Scripts/activate

# 2. 실행
python coupang_scraper_final.py
```

---

## 💡 팁

- 가상환경이 활성화되면 프롬프트 앞에 `(venv)`가 표시됩니다
- `py` 명령어가 안 되면 `python` 또는 `python3`를 사용하세요
- 패키지 설치가 완료되면 다음번부터는 가상환경 활성화만 하면 됩니다
