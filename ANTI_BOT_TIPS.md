# Google 봇 감지 우회 방법

## 현재 적용된 우회 방법

1. **봇 감지 속성 제거**
   - `navigator.webdriver` 속성 숨김
   - 자동화 플래그 제거

2. **자연스러운 브라우저 동작**
   - 랜덤 대기 시간 (5-10초, 6-12초)
   - 자연스러운 스크롤 패턴
   - 마우스 움직임 시뮬레이션

3. **User-Agent 랜덤 선택**
   - 여러 User-Agent 중 랜덤 선택
   - 실제 브라우저처럼 보이도록 설정

4. **캡차 감지 시 사용자 개입**
   - 브라우저가 열려 있으면 수동으로 캡차 해결
   - 해결 후 Enter 키를 누르면 계속 진행

## 추가 해결 방법

### 1. 대기 시간 증가
현재 대기 시간을 더 늘려서 자연스럽게 만들 수 있습니다:
- 페이지 간 대기: 10-20초
- 요청 간 대기: 5-10초

### 2. 검색 빈도 줄이기
- 하루에 여러 번 실행하지 않기
- 실행 간격을 최소 1시간 이상 두기

### 3. VPN 사용
- 다른 IP 주소에서 실행
- 특히 한국 IP가 아닌 경우 감지될 수 있음

### 4. 브라우저 프로필 사용
- 실제 사용자 프로필 사용
- 쿠키 및 세션 정보 활용

### 5. 수동 개입
- `headless=False`로 설정하여 브라우저가 보이도록
- 캡차 발생 시 수동으로 해결
- 캡차 해결 후 Enter 키를 누르면 계속 진행

## 캡차 발생 시 대응

1. **브라우저 창 확인**
   - 브라우저가 열려 있으면 캡차 화면이 보임
   - 캡차를 수동으로 해결

2. **대기 시간**
   - 캡차 해결 후 5-10초 대기
   - 자동으로 계속 진행

3. **재시도**
   - 캡차가 계속 발생하면 잠시 대기 후 재시도
   - 또는 다른 시간대에 시도

## 주의사항

- Google의 봇 감지는 매우 정교하므로 완전히 우회하기는 어렵습니다
- 과도한 요청은 계정 차단을 유발할 수 있습니다
- 적절한 간격을 두고 실행하세요
- 수동 개입이 필요한 경우가 많습니다
